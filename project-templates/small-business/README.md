# قالب موقع شركة صغيرة

قالب موقع بسيط واحترافي باللغة العربية، مناسب للشركات الصغيرة ومكاتب الخدمات.

## المميزات
- تصميم متجاوب (يعمل على الجوال، التابلت، الكمبيوتر)
- هيكل HTML5 ونمط CSS عصري
- سهل التخصيص والتعديل

## كيفية الاستخدام
1. حمل الملفين `index.html` و `style.css`
2. ضعهما في نفس المجلد على الاستضافة لديك
3. عدّل النصوص والصور حسب حاجتك
